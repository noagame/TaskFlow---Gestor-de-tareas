# TaskFlow - Gestor de Tareas

Proyecto web simple para gestionar tareas usando HTML, CSS (Bootstrap), JavaScript ES6+ y jQuery.

Estructura:
```
/front
  index.html
  /scripts
    script.js
objetivo.txt
README.md
```

Para ejecutar: abre `front/index.html` en un navegador.

Repositorio en GitHub: https://github.com/noagame/TaskFlow---Gestor-de-tareas
